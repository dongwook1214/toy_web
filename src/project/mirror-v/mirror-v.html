Mirror-V

앱기능

의료 분야 연구를 위한 헬스 케어 앱으로 자신의 목소리를 가진 인공지능에게 상담받을 수 있으며, 상담 예약, 푸시 알림, 기록 확인 등의 기능 또한 가지고 있습니다.

개발 스텍

  • Flutter
  • Firebase
  • Clova API

대학교에 들어와 처음으로 진행하게 된 팀 프로젝트로 mirror voice , me love 라는 뜻을 가지고 있습니다. 의료 분야 연구를 위한 헬스 케어 앱으로 자신의 목소리를 가진 인공지능에게 상담받으면 어떤 효과가 있을지 알아보기 위해 만들어졌습니다. 팀구성 인원은 2 명이었으며 저는 개발과 관련된 일들을 맡았습니다. 이 프로젝트에서 API 서버에 오디오 파일을 요청하고 보내는 기능을 많이 사용하며 비동기 처리 ,http package , 바이트 데이터 등을 다루는 능력을 키울수 있었습니다.
